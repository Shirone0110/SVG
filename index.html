What is scale?

Scale is a function that is used to convert a dataset in a specific domain into a different data set in a specific range. We can use scaleLinear, scaleSqrt, scalePow to scale data linear, square root, or exponential. Since the unit of the website is pixel, and unit of whatever out data is would probably not pixel, so we need something to convert betweem those units, and it's scale.

Why is it useful?

Scale can be used to transform data set with very big values to smaller values while keeping the scale of the original data set in order to express the data into visualizations.
Another use of scale is it has a built-in function .clamp() which can help ensure the input data is valid, so our program won't crash.
